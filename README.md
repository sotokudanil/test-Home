# Flip Mobile Registration Automation Testing


## Project Description

This project contains mobile automation testing for Flip registration flow using WebdriverIO and Appium.


## Technology

- WebdriverIO
- Appium
- TypeScript
- Mocha
- Android Emulator


## Test Scenario

### Positive Test

1. Register using Phone Number
2. Register using Email


### Negative Test

1. Verify invalid OTP validation using OTP 111111



## Installation


Clone repository:

git clone <repository-url>


Install dependency:

npm install



## Run Automation


Start Appium server:

appium


Run all test:

npx wdio run wdio.conf.ts
